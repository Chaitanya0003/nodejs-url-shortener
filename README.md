A lightweight URL shortener service built with **Node.js**, **Express**, and **MongoDB**. It generates 8-character short IDs for any URL and automatically tracks visit analytics on every redirect.

---

## Features

- Generate a short URL from any long URL
- Redirect users from a short URL to the original destination
- Track every visit with a timestamp
- Retrieve total click count and full visit history per short URL

---

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18+
- [MongoDB](https://www.mongodb.com/) running locally on the default port (`27017`)

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/nodejs-url-shortener.git
cd nodejs-url-shortener

# Install dependencies
npm install
```

### Run the Server

```bash
npm start
```

The server starts on **http://localhost:8001**

## API Reference

### 1. Shorten a URL

**POST** `/url`

**Request Body (JSON)**

```json
{
  "url": "https://www.example.com/some/very/long/path"
}
```

**Response `201`**

```json
{
  "shortId": "aB3dEf9Z"
}
```

---

### 2. Redirect to Original URL

**GET** `/:shortId`

Redirects the client to the original URL associated with the given `shortId`. Every visit is recorded with a timestamp.

**Example**

```
GET http://localhost:8001/aB3dEf9Z
‚Üí 302 Redirect ‚Üí https://www.example.com/some/very/long/path
```

---

### 3. Get Analytics

**GET** `/url/analytics/:shortId`

Returns the total number of clicks and the full visit history for a short URL.

**Response `200`**

```json
{
  "totalClicks": 3,
  "analytics": [
    { "timestamp": 1716400000000 },
    { "timestamp": 1716400500000 },
    { "timestamp": 1716401000000 }
  ]
}
```

## Database Schema

**Collection:** `urls`

| Field          | Type     | Description                        |
|----------------|----------|------------------------------------|
| `shortId`      | String   | Unique 8-character ID (nanoid)     |
| `redirectUrl`  | String   | The original long URL              |
| `visitHistory` | Array    | Array of `{ timestamp: Number }`   |
| `createdAt`    | Date     | Auto-generated by Mongoose         |
| `updatedAt`    | Date     | Auto-generated by Mongoose         |
